# vue3-admin-typescript-vite
**Live demo:** https://HaitaoWang555.github.io/vue3-admin-typescript-vite

## Started
```bash
# install pnpm
npm i -g pnpm

# install dependency
pnpm install

# develop
pnpm dev
```

## Build

```bash
# build for test environment
pnpm build:stage

# build for production environment
pnpm build:prod
```
